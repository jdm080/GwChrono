# GwChrono
A timing plugin for GWtoolbox!


[Quick Download](https://github.com/jdm080/GwChrono/raw/refs/heads/main/GWChronoV1.dll)
# Quick Start
A concise walkthrough to get GWChrono installed, loaded, and saving runs.

Download the plugin DLL.

Copy the DLL to:

Documents\GWToolboxpp\<PCName>\plugins

Launch GWToolbox (Toolbox).(or refresh if you already had it open). Open Toolbox → Settings → Plugins and Load GWChrono.

Complete a run you want to save. Click Complete, enter a name, then Save. (you won't see any templates until you do your first save/complete.

# Saving Runs and Templates
Save a run

After finishing a run: Click Complete → type a name → Save.

Use a template

Left click a template to set it active.

When a template is active, saving will use that template automatically.

Right click a template to unset it if you need a different setup.

Template behavior summary

Left click = set active (quick reuse). it will stay active until you unset! 

Right click = Unset and Delete

If you want to save to the same template again, set it active first. If you messed up naming it browse to the file path and rename, or you can delete it al together and start over.

# UI Customization
Click the = in the top right of the Chrono window → Chrono Window Editor.

Running and Leveling each have their own layout and settings.

Anchors and layout: Assign values to anchors using the dropdowns.

Drag anchors by the small circle attached to each element to reposition them.

Extensive color and text customization is available — fonts, colors, visibility, and more.

Save layouts per profile so switching profiles restores the correct overlay.
